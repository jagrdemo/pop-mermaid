Risanje diagramov s knjižnico Mermaid

Mermaid je JavaScript knjižnica, ki omogoča enostavno definicijo različnih tipov diagramov. Te lahko preprosto vključimo v HTML ali Markdown dokument. Diagrame lahko pripravimo tudi v spletnem urejevalniku in jih izvozimo kot slike.

Osnovni diagram

Diagram tipa "flowchart" je sestavljen iz vozlov in povezav med njimi.

flowchart LR Z(Začetek) P((Proces)) K(Konec) N{"`**Napaka**`"} Z==>P==>K Z-.->K Z & P-->|ups|N style N fill:red
